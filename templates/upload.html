Create an interactive Similarity Heatmap from Text Data.

This system uses Scikit-Learn Cosine Similarity and the TfIdfVectorizer() function with stopwords removed to analyze the similarity between different categories of text.
D3.js is used to create an interactive similarity matrix, or heatmap, like this:

All you have to do is upload a labelled csv file!

Upload CSV File:

The file must be 2 colums, with no headers (column names)

The first column will be the label, or category, of the text

The second column will be the text to analyze

Labels may repeat. Text from repeating labels will be combined into one document for that label

For example: a CSV file containing Yelp reviews across an array of restaurants might have cuisine type as the label, and an individual review in the text column.
